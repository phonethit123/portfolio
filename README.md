# Portfolio

This is a modern Next.js portfolio website, styled with Tailwind CSS and ready to deploy on Vercel.

## Features
- Next.js App Router
- Tailwind CSS styling
- ESLint for code quality
- Projects section (customize in `/src/app/page.js`)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view your site.

## Deploy
- Push your code to GitHub and import the repo at [vercel.com](https://vercel.com/).
- Vercel will auto-detect Next.js and deploy your site.

## Customization
- Edit `/src/app/page.js` to update your homepage and projects.
- Update `/src/app/globals.css` for global styles.
